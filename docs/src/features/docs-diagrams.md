# Docs & Diagrams

Beyond simply generating tasks, Plan AI acts as an automated technical writer for your engineering team. As your team discusses architecture changes, database schemas, or system flows during meetings, Plan AI captures those decisions.

## Automated Technical Documentation

You can use Plan AI to instantly draft documentation based on a meeting. 

If your team spends 45 minutes debating a new microservice architecture, you don't need to assign someone to write the internal wiki page.

1. Open the **Chat** interface for the specific meeting.
2. Type: *"Draft a technical specification document based on our microservice discussion."*
3. The AI will output a fully formatted Markdown document detailing the goals, the chosen tech stack, the rejected alternatives, and the agreed-upon API contracts.

You can then copy this directly into Notion, Confluence, or your GitHub repository.

## System Diagrams (Mermaid.js)

Explaining complex data flows with text is difficult. Because Plan AI understands technical context, it can automatically generate architecture diagrams using [Mermaid.js](https://mermaid.js.org/).

In the chat interface, simply ask:
> *"Map out the data flow we just agreed upon for the new authentication system using a Mermaid sequence diagram."*

The AI will generate the raw Mermaid code, which will be automatically rendered into a beautiful, visual diagram right inside the chat window. You can easily export this diagram to your technical documentation.

## Generating Slide Decks

If you need to present engineering decisions to non-technical stakeholders (like the CEO or a client), Plan AI can do the heavy lifting.

Navigate to the **Presentations** tab to convert meeting transcripts into slide decks. The AI will summarize the technical jargon into high-level business goals, bullet points, and key takeaways perfectly formatted for a presentation.
