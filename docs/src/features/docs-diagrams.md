# Docs & Presentations

Beyond simply generating tasks, Plan AI acts as an automated technical writer for your engineering team. As your team discusses architecture changes, database schemas, or system flows during meetings, Plan AI captures those decisions.

## Automated Document Generation

You can use Plan AI to instantly draft comprehensive documentation based on a meeting.

If your team spends 45 minutes debating a new microservice architecture, you don't need to assign someone to write the internal wiki page. There are two ways to generate documentation:

**1. Automated Generation (Recommended)**
When finishing a recording on your Desktop or Mobile app, simply toggle the **"Generate a document"** switch in the post-recording configuration screen. Plan AI will automatically process the meeting and output a fully formatted Markdown document detailing the goals, chosen tech stack, rejected alternatives, and agreed-upon API contracts.

**2. On-Demand via Chat**
If you forgot to toggle the switch, you can always open the **Chat** interface for the specific meeting and type: _"Draft a technical specification document based on our microservice discussion."_

You can easily copy this output directly into Notion, Confluence, or your GitHub repository.

## System Diagrams (Mermaid.js)

Explaining complex data flows with text is difficult. Because Plan AI understands technical context, it can automatically generate architecture diagrams using [Mermaid.js](https://mermaid.js.org/).

In the chat interface, simply ask:

> _"Map out the data flow we just agreed upon for the new authentication system using a Mermaid sequence diagram."_

The AI will generate the raw Mermaid code, which will be automatically rendered into a beautiful, visual diagram right inside the chat window. You can easily export this diagram to your technical documentation.

## Generating Slide Decks

If you need to present engineering decisions to non-technical stakeholders (like the CEO or a client), Plan AI can do the heavy lifting by converting meeting transcripts into slide decks. The AI summarizes the technical jargon into high-level business goals, bullet points, and key takeaways perfectly formatted for a presentation.

**How to generate slides:**

- **Automatically:** Toggle the **"Generate slides"** switch in the post-recording configuration screen before saving the meeting.
- **On-Demand:** Navigate to the **Presentations** tab of any saved meeting to generate them manually.
